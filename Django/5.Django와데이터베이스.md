# Django와 데이터베이스

### 데이터베이스

* ORM(Object Relation Mapping): 

  * sql언어 없이 파이썬을 이용해서 데이터베이스를 조작 가능

* 데이터베이스

  * row와 column으로 구성되어있는 테이블과 비슷함

* 파이썬의 Class

  * 객체지향적 프로그래밍

  * 객체를 만드는 틀의 역할

  * ```python
    # 파이썬을 이용한 데이터베이스 생성 예시
    class Blog:
        Id = "숫자"
        제목 = "문자"
        본문 = "문자"
        날짜 = "날짜"
    ```
