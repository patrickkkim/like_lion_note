## HTML 2

* HTML은 꾸미는 언어가 아니다! -> CSS가 담당

* HTML 코드

  * DOCTYPE: **HTML로 작성됐다고 알리는 태그**
  * HEAD: **화면에 등장 안하는 문서 설명 태그(Title, 인코딩 방식 등)**
  * BODY: **문서에 보이는 태그(h1, p, li, 등)**

* 예시:

  ```html
  <!DOCTYPE html> <!-- html 명시 -->
  <html> 			<!-- html 시작 -->
      <head></head> <!-- 문서 중요 정보 -->
      <body></body> <!-- 보이는 내용 -->
  </html>
  ```
