# Bootstrap

### 소개 및 시작

* 중복 CSS 코드 줄이는 법:

  * 정의한 스타일을 상속해서 바꾸고 싶은 부분은 덮어쓰면 된다:

    * ```css
      .btn {
          cursor: pointer;
          border: none;
          padding: 4px 20px;
          background: red;
          color: white;
      }
      
      .btn-login {
          border: 2px solid black;
          background: black;
      }
      ```

  * 이 CSS를 적용하기:

    * ```html
      <button class="btn btn-login">로그인</button>
      ```

* **부트 스트랩:** 오픈 소스 프런트엔드 프레임워크

  * 헤드 태그의 링크 태그에 삽입

<br>

### 다양한 기능들

* 폼과 버튼
*  그리드 시스템 
  * 바둑판에 두듯 배치 가능
  * container를 이용해서 안에 row, col를 집어 넣음
  * 그리드는 정확히 12개의 col를 가짐
  * 반응형 웹 구현도 가능
    * lg, sm, md, xl 등을 이용함
  * ```.col-lg-4```
    * lg: 기준이 되는 화면의 사이즈
    * 4: 12등분 중 차지할 비율
* 충돌 없이 부트 스트랩 프로퍼티 덮어주는 방법
  * 부트 스트랩은 ```.class``` 클래스 선택자 사용한다
  * ```!important```, 아이디 선택자, 클래스 타입 선택자(div.class) 사용해서 덮어주자
* 더 다양한 기능들을 검색해서 찾자!!
